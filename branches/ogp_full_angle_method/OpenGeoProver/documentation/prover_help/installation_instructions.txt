1. Copy OpenGeoProver.jar file together with installation script for
   specific OS, in desired directory.

2. For UNIX-like system make sure to set permissions for executing installOGP script.

3. Run installOGP script. This will prepare the directories for using OpenGeoProver.

4. In scripts directory there is runOGP script for running prover.

5. In order to be able to run the prover, there must be installed Java in computer
   and PATH and CLASSPATH environment variables set.
